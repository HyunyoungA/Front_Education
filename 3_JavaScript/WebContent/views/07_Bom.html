BOM(Browser Object Model)

navigator객체

웹페이지를 실행하고 있는 브라우저에 대한 정보를 갖고 있는 객체

screen객체

웹브라우저 화면이 아닌 운영체제 화면(모니터)의 정보를 가지는 객체

location객체

브라우저 주소 표시줄과 관련된 객체

location.href

페이지 이동


reload()

페이지를 새로고침하는 메소드

assign()과 replace

두 메소드 모두 location의 href를 재지정하는 메소드(페이지 이동)

history객체

브라우저 방문기록과 관련된 객체
